## Description

The Portfolio Website is a visually appealing and responsive site that highlights the author's skills, work, and provides a way to get in touch. It includes the following elements:

- A "Main" section with a greeting, name, and a call to action button.
- An "About" section that introduces the author and their passion for technology.
- An image illustrating the author's work or interests.
- A "Contact" section with the author's name, title, and a message encouraging visitors to get in touch.

## Features

- A simple and clean design with a clear call to action.
- Utilizes the Bootstrap framework for responsive design and styling.
- Presents the author's name and profession prominently.
- Encourages interaction by offering a "Work With Me" and "Contact Me" button.
